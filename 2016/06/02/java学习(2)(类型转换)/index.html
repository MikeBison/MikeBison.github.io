java学习(2) 类型转换

2016-06-02 • 分类: -java • 标签: -java -学习笔记

java学习(1)

声明:仅供个人巩固学习使用.

类型转换

不同数据类型转换可能损失精度

java计算中会发生自动类型提升

示例:

byte b = 3;
b = (byte)(b + 2);

将b+2的和强制转换为byte类型

一些坑

linux系统中的转移字符是一个字符

windows中的转义字符是两个字符

short s = 3;
s = s + 5

运算了俩次,第一次为s+5
第二次为给s赋值,由于和的类型补位short,所以编译失败

s += 3;

运行了一次,编译通过

单双与号区别:

双与有短路运算.

一个int在内存中占4个8位

一个数左移计算为2的位移次方乘以这个数,右移相反,位移是运算速度最快的,右移最高位补什么由原有最高位决定,原来为0则补0,为1则补1,无符号右移最高位永远补0

同一个数异或俩次还是原数

俩个数换位:

int n=3,m=4;    
n = n^m;
m = n^m;//m = n^m^m;//原n
n = n^m;//n = n^(n^m);//原m
        //n = 4;m = 3;
