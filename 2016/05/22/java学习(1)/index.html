声明:仅供个人学习使用.

dos常用命令

cd   //转到目录
md   //创建目录
dir  //列出当前路径下所有目录
rd   //删除目录
cd/  //转到当前根目录
cd.. //转到上级目录
del  //删除文件
del  //*.txt删除所有.txt文件
exit //退出dos
set  //配置环境变量

java名词解释

jvm java运行环境,即虚拟机,java能够夸平台就是因为可以在不同的平台安装虚拟机进行运行,类似于js中的v8引擎.

jdk java开发工具包(包涵了开发工具和运行环境)

jre java开发运行环境

一些坑

在执行过

set classpath = xxx/xxx;

后,虚拟机执行文件时先找到classpath,然后再找所在目录文件

set classpath = xxx/xxx

带分号先找classpath再找当前目录,带分号先找classpath,再找当前目录,区别在于如果没有会报错,一般set classpath不带分号,防止出错.

记录

  • main函数可以保证java类的独立运行,是一个程序的入口,jvm会调用它
  • 类名需要大写(与js中的构造函数相同,其实js中的构造函数借鉴于其他语言的类,主要借鉴了java),这是通用的语言规范

总结

本日学习了java术语的一些简介,环境配置等内容
